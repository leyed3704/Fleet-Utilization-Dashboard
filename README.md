# Oladipupo Store Data Analysis (Interactive Dashboard creation )
## Project Objective
Oladipupo Store wants to create an annual report of Fleet utilizaytion for 2022, 2023 and 2024, to evaluate the utilization of its fleets.
## Dataset used
https://docs.google.com/spreadsheets/d/1CAT5kcz5HGAEOz4lmiokEmLFbbs6cQQI/edit?usp=drive_link&ouid=104177868748180292987&rtpof=true&sd=true
## Process
- Verify data for any missing values and anomalies, and sort out the same .
- Made sure data is consistent and clean with respect to data type, data format and values used.
- Created tables and charts according to the questions asked.
- Merge all  tables into one dashboard and apply slicer to make dynamic.
## KPIs
- Total Miles
- Total Revenue Total Fuel Cost
- Total Revenue
- Total Maintenance Cost.

  ## Project Insight  
- The fleet covered a total of 122M miles, generating 298.62M in revenue.
- Fuel cost is significantly high at 95.59M, making it one of the largest operational expenses.
- Volvo appears to generate the highest revenue among all truck makes.
- Idle hours are distributed across all truck brands, with Volvo showing the highest share.
## Conclusions
- The fleet operation is generating strong revenue despite high fuel expenses.
- Volvo trucks contribute the most to overall fleet performance.
- Reducing idle hours and fuel consumption could improve overall operational efficiency and profitability.
